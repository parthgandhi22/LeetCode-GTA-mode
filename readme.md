# 🎮 LeetCode GTA Mode

Turn your LeetCode submissions into GTA moments.

Get the iconic **Mission Passed! Respect +99** screen on Accepted submissions and **Wasted** on failed submissions, complete with GTA-style sounds and overlays.

![LeetCode GTA Mode](./icon128.png)

---

## ✨ Features

- 🟩 **Mission Passed!** overlay on Accepted submissions
- 🟥 **Wasted** overlay on Wrong Answer, TLE, RE, MLE, etc.
- 🔊 GTA-inspired sound effects
- 🌫️ Cinematic blur and popup animations
- ⚡ Lightweight and fast
- 🔒 No tracking
- 🔒 No data collection
- 🔒 No permissions required

## 📦 Installation

### Method 1: Load Unpacked (Recommended)

1. Download or clone this repository:

```bash
git clone https://github.com/parthgandhi22/LeetCode-GTA-mode.git
```

2. Open Chrome and go to:

```
chrome://extensions
```

3. Enable **Developer mode**.

4. Click **Load unpacked**.

5. Select the extension folder.

6. Open LeetCode and submit a solution.

Enjoy the GTA vibes! 🚀

## ⚙️ How It Works

1. The extension injects a script into LeetCode pages.
2. It hooks into LeetCode's submission status requests.
3. It detects:
   - Accepted submissions
   - Wrong Answer
   - Runtime Error
   - Time Limit Exceeded
   - Memory Limit Exceeded
4. It displays GTA-themed overlays and plays sounds accordingly.
